## DataTypes

1. **_Char_**

   - allows any character (max upto 255 characters)
   - fixed length
   - default width 1
   - wastage of HD space

2. **_Varchar_**
   - allows any character (max upto 65,535 characters) (64KB-1)
   - variable length
   - no default width (width has to be specified)
   - conserve on HD space
